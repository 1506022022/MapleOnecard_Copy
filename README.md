## 프로젝트 소개
**'메이플스토리'** 의 미니게임 **'메이플 원카드'** 를 모작한 개인 프로젝트입니다.

해당 포트폴리오에 사용된 리소스는 모두 메이플스토리의 **허락을 받고** 사용하였습니다.

플레이 영상 : <https://youtu.be/q9gXEUtKlwg>
